Projects

Most recent work

Animal Detection in Video Using YOLOv5 and Real-time Playback

This project leverages the YOLOv5 deep learning model to detect and identify animals (dog, cat, horse, sheep, and cow) in video files. It provides real-time video playback, displaying bounding boxes and confidence labels for detected animals. Additionally, the system plays an alert sound when an animal is detected, with the option to stop detection at any point through a user interface.

GitHub Repository

Real-Time Emotion Detection Using Webcam and Tkinter

This program uses OpenCV and Keras to detect facial emotions from a live webcam feed. It employs a pre-trained convolutional neural network (CNN) model to predict emotions such as "Happy," "Sad," or "Angry." The real-time video is displayed in a resizable Tkinter window, where the frame dynamically adjusts to the window size while annotating detected emotions on faces.

GitHub Repository

Email Spam Filter

The Email Spam Filter project uses machine learning to classify emails as spam or non-spam (ham). It employs a Naive Bayes classifier, trained on labeled email data, to make real-time predictions. The project features a user-friendly GUI for easy interaction and stores classified emails in separate folders for future reference.

GitHub Repository

Stock Price Prediction Using XGBoost

This project uses machine learning techniques, specifically XGBoost, to predict stock prices based on historical data and technical indicators. It employs various features such as moving averages, RSI, MACD, and Bollinger Bands to train the model. The application provides real-time predictions for stock prices (Open, High, Low, Close) and visualizes them through interactive charts.

GitHub Repository

Vehicle Speed Detection System

This project utilizes computer vision techniques to estimate the speed of vehicles in a video. It processes the video frames to detect moving objects, calculates their speed based on their position changes between frames, and identifies vehicles exceeding a defined speed limit. If the speed limit is exceeded, images of the violating vehicles are captured and stored in a specified folder for further action.

GitHub Repository
